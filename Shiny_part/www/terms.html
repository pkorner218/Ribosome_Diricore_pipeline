Terms and Conditions for the “Usage” of Bumpfinder

Last updated [day month year]



The Bumpfinder website and the server are hosted in the Netherlands at the Netherlands Cancer Institute (NKI) and is regulated by the European General Data Protection Regulation (GDPR). In compliance with these regulations we hereby provide information on the data processing pipeline as well as the terms and conditions under which data is handled by the Bumpfinder server.

Section 1. Data collection

- Email address (1)

- Samples containing ribosomal sequencing data (2)

- Information on the sample preparation, namely the used adapter sequences and the species of origin (3)

- Text files with additional treatment information on the samples (4)

- IP address (5)

- Time and Date the analysis was started and finished (6)

Email address (1), samples containing ribosomal sequencing data (2) and the text files (3,4) are uploaded by the user himself.
IP addresses (5) as well as start and end time of the analysis (6) are collected automatically at the initiation of Bumpfinder workflow by the user.

Section 2. Data usage

- Samples and sample specific information such as preparations and treatments (2,3,4) are required to run the Bumpfinder analysis workflow (refer the manuscript).

- Email addresses (1) are collected to transfer results in HTML format (ref) to the user and to notify users of any issues.

- IP address (5,6) and time and data of the analysis is used to create a log on the server activity and to monitor server traffic.

Section 3. Data handling and storage

All uploaded data are stored on separate servers (server addresses..) of the NKI. Access to the server is limited to authorized personal (list of authorized personnel at the end of the document) and further secured by security measures such as 2factor authorization.

Uploaded data will automatically be analyzed and no attempt to view results will be done by the authorized personnel.

All the analysis will be stored during the analysis period and one week afterwards. After the results have been transferred to the user, all information will be deleted within one week.

The log file including IP address, time and date and time of the analysis will remain stored.

All collected data will at no point be shared, or made accessible for download, copying, editing or viewing to authorized personnel, third parties inside or outside of NKI.

Section 4. User rights under the GDPR and their implementation

- The right to access and request copies of your personal data. During the analysis period and 7 days after delivery of results, users can request to receive their data including log files. The infrastructure setup for receiving the data lies at the disposal of the user.

- The right to request corrections on any data you provided.

- The right to have your data transferred back to you. As all results are sent to the user as email we comply with this automatically.

- The right to object any processing. If a user does not want the mentioned data to be collected and processed he can contact ... and request the scripts. This will allow the Bumpfinder analysis without data collection.

- The right to have any data you gave erased. All data will be erased automatically … hours after a run is complete. A user can request his data is erased earlier by contacting … .

Section 5. User obligations

The user has to ensure that he does not upload any sensitive information regarding his own privacy, the privacy of others or the privacy of sample origins.

Sample names should be anonymized by the user prior to uploading should the data contain any sensitive information.

The user has to ensure he has the rights and permissions to own and share the sample data and to have it analyzed.

The user has to ensure he is not violating intellectual property regulations by uploading and analyzing the data.

In the case any plots or information that arrive from the Bumpfinder are used or shared, the user is obliged to cite the Bumpfinder manuscript as follows: [.....]

Section 6. Remark

The Bumpfinder software is a consistent pipeline. It is not possible to have parts of it altered or additional analysis to be run through the webserver. In the case any of this is desired, please contact authorized personnel (Section 7) for an official collaboration.
